# 🛒 Ecommerce API

A powerful, modular, and secure E-commerce backend built with **NestJS (v11)**, **PostgreSQL**, and **TypeORM**.

## 🚀 Overview
This project provides a robust foundation for an online marketplace, featuring production-ready practices such as JWT authentication, data validation, and automated API documentation.

## 🛠 Tech Stack
* **Framework:** [NestJS](https://nestjs.com/) (Node.js)
* **Language:** TypeScript
* **Database:** PostgreSQL
* **ORM:** TypeORM
* **Security:** Passport.js (JWT Strategy) & Bcrypt
* **Documentation:** Scalar & Swagger UI
* **Validation:** Class-validator & Joi

## 📋 Features
- [x] **Modular Architecture:** Highly scalable and organized code.
- [x] **Authentication:** Secure Login/Register using JWT.
- [x] **Security:** Password hashing and protected routes.
- [x] **API Docs:** Interactive documentation with Scalar (accessible at `/docs`).
- [x] **Testing:** Comprehensive testing suite with Jest.
- [x] **SEO Friendly:** URL Slug generation for products.
